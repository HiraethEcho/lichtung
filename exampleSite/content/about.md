# 关于本站

这是 **Lichtung** Hugo 主题的示例站点。

## 主题特性

Lichtung 是一个简洁而功能丰富的 Hugo 主题：

- **五个分类系统**：标签、分类、系列、主题、体裁
- **四种列表样式**：plain（卡片）、date（归档）、list（简约）、split（分页）
- **评论系统**：支持 Waline、Giscus、CWD 多种评论
- **交叉链接**：自动追踪文章间的出链和入链
- **搜索功能**：基于 Fuse.js 的客户端全文搜索
- **暗色模式**：支持系统偏好和手动切换
- **热力图**：展示一年内的发文情况
- **数学公式**：MathJax 支持
- **Mermaid 图表**：代码块绘制图表
- **字数统计**：多维度统计站点文章字数

## 分类系统

本站使用 Hugo 的 taxonomy 系统进行内容分类：

| 分类       | 说明 | 示例             |
| ---------- | ---- | ---------------- |
| tags       | 标签 | 入门、示例、主题 |
| categories | 分类 | 教程、杂谈       |
| series     | 系列 | 示例系列         |
| topics     | 主题 | 工业技术生活     |
| genres     | 体裁 | 议论、叙事       |
